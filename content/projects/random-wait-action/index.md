A GitHub Action that introduces configurable random delays into CI/CD
pipelines. It is designed for use cases where rate limiting, distributed
execution timing, or non-deterministic scheduling is required in automated
workflows. The action is lightweight, fully configurable, and intended for
integration into larger automation systems rather than standalone use.
