A minimal, privacy-preserving search redirection tool that enforces direct
delegation to external search engines. JFGI provides a programmatic and
human-readable interface for converting arbitrary queries into structured
web search requests, reducing conversational dependency for factual retrieval
tasks. The system is designed as a lightweight utility for information
discipline, emphasizing reproducibility of search behavior, reduction of
cognitive overhead in query formulation, and adherence to external knowledge
systems rather than internal speculation.
