A Bayesian-network based framework for modeling kinase signaling relationships using
PamStation kinome array signals. KINNET utilizes Bayesian Networks to infer
the inter-kinase relationships from the PamChip Kinome Array results.
The system is designed for modular analysis of signaling networks and facilitates
exploration of kinase-driven biological processes in a reproducible
computational environment.

This project was part of my Masters Thesis research. You can
read more about it in
[my thesis.](http://rave.ohiolink.edu/etdc/view?acc_num=mco1628385958319609)
