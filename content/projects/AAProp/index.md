A lightweight Rust-based REST API that provides structured access to
physicochemical properties of the 20 standard amino acids. It is designed for
fast, programmatic use in bioinformatics workflows, supporting downstream
applications in protein analysis, sequence processing, and computational
biology toolchains. The project emphasizes simplicity, speed, and ease of
integration into larger systems.
